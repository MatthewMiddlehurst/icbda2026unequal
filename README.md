# icbda2026unequal

Repo for the ICBDA 2026 unequal length time series submission. More to come.

Data:
    datasets.py can be used to process the .ts files uploaded by the UCR archive maintainers (https://zenodo.org/records/11198697) into unequal length versions.
    Outside of blind reviews, we will upload the data itself for ease of use.

Results:
    .csv files are provided which give the accuracy on the mixed unequal archive
